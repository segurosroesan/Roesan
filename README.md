# Roesan Web App

Aplicación web oficial de Roesan Seguros, construida con Next.js (App Router), React, Tailwind CSS y conectada a InstantDB.

## 🚀 Repositorio y Despliegue Oficial (Abril 2026 en adelante)

**¡IMPORTANTE!** A partir de abril de 2026, el repositorio oficial y la cuenta de despliegue pertenecen al cliente final.

- **Repositorio Git:** `https://github.com/segurosroesan/Roesan.git`
- **Despliegue (Hosting):** Netlify (Cuenta de Roesan)
- **Rama Oficial:** `master`

> ⚠️ **Atención:** Las cuentas anteriores de `Grupo Moraki / jorgeH` ya no deben usarse para gestionar el código fuente ni el despliegue de este proyecto.

## 🛠 Entorno de Desarrollo

Para iniciar el servidor de desarrollo local:

```bash
npm run dev
# o
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) (o el puerto que asigne Next) en tu navegador para ver el resultado.

## 🧱 Stack Tecnológico

- **Frontend:** Next.js (16.x) App Router con React 19
- **Estilos y UI:** Tailwind CSS v4, Framer Motion, Anime.js
- **Base de Datos & Backend:** InstantDB (Real-time DB)
- **Formularios:** React Hook Form + Zod
